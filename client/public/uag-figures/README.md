# UAG (Part 107) Figures

All figures for Part 107 / UAG exam questions go in this folder ONLY.
Do NOT use figures from par-figures/, ira-figures/, cax-figures/, or atp-figures/.

FAA figure names often repeat across exams (e.g., "Figure 1", "Figure 2").
Keeping them in separate folders prevents the wrong figure being shown.

## Naming convention
Use descriptive names, not just figure numbers:
  uag-sectional-class-b.jpg
  uag-sectional-class-d.jpg
  uag-metar-sample-1.jpg
  uag-taf-sample-1.jpg
  uag-density-altitude-chart.jpg
  uag-airport-diagram-sample.jpg
  uag-sectional-legend-obstruction.jpg
